## Installations

* Clone and `npm i`
* Set your .env.local file with the following configuration found on facebook project and firebase projects
    *   ```FACEBOOK_CLIENT_ID=""
            FACEBOOK_CLIENT_SECRET=""
            NEXTAUTH_URL=http://localhost:3000/
            FIREBASE_API_KEY=""
            AUTH_DOMAIN=""
            PROJECT_ID=""
            STORAGE_BUCKET=""
            MESSAGING_SENDER_ID=""
            APP_ID=""```
